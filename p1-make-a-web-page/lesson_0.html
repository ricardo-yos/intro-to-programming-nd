A Review of the Basic Concepts of HTML

A Brief Definition of HTML

The world wide web is basically a collection of HTML documents. HTML is the language in which these documents are made. HTML stands for Hypertext Markup Language. Through the HTML we can add images, videos, texts and other sources that compose the internet. These HTML pages are linked together with hyperlinks.

Web Operation

The main idea of the web operation is the communication between the user’s computer and the servers. A person when accessing a page using a web browser initiates communication to a server by requesting a specific resource using a HTTP Protocol. The servers are computers which store and process information that make up the internet. The server responds to the user with the content of the requested resource in a HTML document, the web browser interprets and displays this document on the user’s screen. For other information, consult Wikipedia page.

Basic HTML

The HTML is composed by the text content which is what you see and the markup which describes how the content looks like. In order to build a HTML page, we need to use HTML elements. A HTML page contains several types of HTML elements. Each element has a specific function on a page and it is usually written in a HTML syntax called tags. Most elements are written with an opening tag and a closing tag, with the tag content between them. This tag structure is represented as follows:

<Name> Contents </Name>

A HTML tag is formed by the name of the element within the angle brackets. The only difference between the opening tag and closing tag is the slash in front of the name for the closing tag.

It is important to note that computers are stupid. It means that they interpret an instruction literally. Even a small mistake given to a computer by the programmer can bring issues and unexpected actions.

HTML Attributes

An HTML attribute can modify some HTML elements giving it an additional functionality or modifying its default function. The attribute is written within the opening tag as shown bellow:

<Tag attribute=value> Contents </Tag>

  • Tag is the name of the HTML element
  • attribute is the Attribute or the property we want to set
  • value is the value of the property to be set

HTML Block and Inline Elements

HTML elements can be either block or inline. Block elements create an invisible box with height and width around the content. While inline elements are just text without height and width properties.
