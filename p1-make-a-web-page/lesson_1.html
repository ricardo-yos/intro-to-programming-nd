The Structure of a Web Page

HTML Tree-Like Structure

Tree-Like Structure is the manner that the browser interprets the page as a hierarchical structure. It is also called DOM (Document Object Model). Everything in a HTML document is a node. Inside each node can have other nodes and going down into these nodes we can find more nodes and so on. This tree structure resembles a tree, where the branches are HTML elements.

Everything is Boxes!

Every element in a Web Page is a box, even the not-rectangular images. A Web Page contains many boxes with different sizes and locations that can be rearranged in many ways, giving different perspectives to the user.

When we are creating the web page from the layout, we start drawing a box around every visual element. The page may have boxes within boxes within boxes. A good tip for visualize all the boxes inside the page is divide it piece by piece, starting from the big box which is the whole page and work down to the small ones.

Exploring Web Sites

Most of browsers provide some built-in Developer Tools that allow web developers to test and debug their code. Google Chrome browser comes with a great developer tools that provide a deep access to the inner structure of any page in the web.
