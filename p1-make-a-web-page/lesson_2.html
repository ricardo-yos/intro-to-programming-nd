CSS Style and HTML Work Together

Adding CSS Style

The HTML language is responsible for giving the structure and the text content of a page, while CSS is the language used to give style for each HTML element. CSS stands for Cascading Sheet Style. The word Cascading means that a specific rule applied to an element will be applied to all elements inside it until this rule reaches the deeper element by cascading.

Applying CSS

The most used method for applying CSS style in a web page is to link the HTML document to an external CSS file. This method makes the work more organized by separating the CSS code from the structural HTML. It also saves a lot of work because a single CSS file can control the layout of multiple web pages all at once.

The CSS expression used to apply style to an element is given follow:

  • selector is the element we want to style
  • Everything between the curly braces is the rule
  • declaration is each line of the code between the curly braces
  • property is the property we want to style
  • value is the value of the property

The Box Elements

All HTML elements are boxes. It is necessary to understand the components of these boxes to set it properly on the page. Each box has four components:

  1. Content is the content of the box and it sits at the very center
  2. Padding is a layer that clears an area around the content
  3. Border is around the padding and content
  4. Margin is the outer layer
